My personal website, built with Remix & Cloudflare Workers.

## Development

### Setup

Switch to the recommended node version using [nvm](https://github.com/nvm-sh/nvm):

```sh
nvm install
```

Install dependencies:

```sh
npm install
```

### Running

```sh
npm run dev
```

### Linting

```sh
# lint code
npm run lint

# typecheck
npm run typecheck
```
