Personal website, made with React, Remix, TypeScript, Cloudflare Pages and DatoCMS as headless CMS.

## Development

### Setup

Switch to the recommended node version using [nvm](https://github.com/nvm-sh/nvm):

```sh
nvm install
```

Install dependencies:

```sh
npm install
```

### Running

```sh
npm run dev
```

### Linting

```sh
# lint code
npm run lint

# lint & autofix code
npm run lint:fix

# typecheck
npm run typecheck
```
